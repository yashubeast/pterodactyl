# How to setup

## Initial Setup
1. docker compose up -d
2. run this command to make a new user `make new_user`
3. if above command failed run this command then do step 2 again `make sqlfix`

## Creating first Node
1. Click settings-icon (tooltip: admin) at top right
2. Click Locations at left sidebar
3. Create a new location
4. Click Nodes at left sidebar
5. Create a new Node

## Node Settings Guide
- Daemon and SFTP Port: same as docker ports exported
